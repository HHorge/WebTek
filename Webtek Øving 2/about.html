Questions

  1. What does the server in the client server architecture do?

    2. The server stores the information, so that it's available for the clients to download.

  2. What does the client do?

    3. A client is anyone who wants to utilize the information stored on a server.

  3. What is an IP address?

    4. An IP address is a unique number (ID) that everything connecting to a network, e.g. the Internet, gets assigned. Comparable to a phone number.

  4. What is DNS?

    5. Domain Name Service is a sort of phonebook, but instead of connecting phone numbers to names, it connects domain names to IP addresses.
    Making sure that the client entering a domain name gets sent to the right server (unique IP address).

  5. What is the difference between the Internet and WWW?

    6. The Internet is a massive network of computers, that allows any computers connected to the same network communicate with each other.
    There are a lot of different types of connections on the Internet, the World Wide Web being one of them.
    The WWW is the part of the Internet that contains all websites.

  6. Explain what HTML is.

    7. HTML is the markup language used to make this document, and is the standard markup language used online.
