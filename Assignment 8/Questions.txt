2. How would you create a blue circle using the Canvas element? Explain with words or code.
    Assuming a filled blue circle:
        function drawCircle(){
            context.fillStyle = "blue";
            context.beginPath();
            context.arc(100, 100, 50, 0, 2*Math.PI); 
            context.stroke();
            context.closePath();
            context.fill()
        }
